## Installation

We recommend that you use a linux distribution or WSL.

    $ sudo apt update

    $ sudo apt install python3.10-venv make

    $ make clean

    $ make configure

    $ make build

    $ make test

We also recommend that you use VSCode, for which we have prepared some integrations.
